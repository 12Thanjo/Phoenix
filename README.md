# Phoenix Engine

Game / Render engine written in C++ and uses Vulkan.


## Building:

### Dependancies:
- Premake5
- VulkanSDK

#### Additional if on Linux (not supported yet):
- libx11-dev
- libxkbcommon-x11-dev
- libx11-xcb-dev
