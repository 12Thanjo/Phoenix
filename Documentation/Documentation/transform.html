Transform

Functions for manipulating the transform of an object

Position:

entity.getX()

Get the x position of the entity

entity.setX(number x)

Set the x position of the entity

entity.getY()

Get the y position of the entity

entity.setY(number y)

Set the y position of the entity

entity.getZ()

Get the z position of the entity

entity.setZ(number z)

Set the z position of the entity

Rotation:

entity.getRotationX()

Get the x rotation of the entity

entity.setRotationX(number x)

Set the x rotation of the entity

entity.getRotationY()

Get the y rotation of the entity

entity.setRotationY(number y)

Set the y rotation of the entity

entity.getRotationZ()

Get the z rotation of the entity

entity.setRotationZ(number z)

Set the z rotation of the entity

Scale:

entity.getScaleX()

Get the x scale of the entity

entity.setScaleX(number x)

Set the x scale of the entity

entity.getScaleY()

Get the y scale of the entity

entity.setScaleY(number y)

Set the y scale of the entity

entity.getScaleZ()

Get the z scale of the entity

entity.setScaleZ(number z)

Set the z scale of the entity
