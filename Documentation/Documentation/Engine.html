Engine

The Engine class is the main class that you will interact with. Engine provides a number of optionally extendable functions.

class Game : public Phoenix::Engine{
public:
Game() : Phoenix::Engine( {"Game Name", 500, 300} ) {};

// I'm optional
void create() override{
// I run when the engine is created
}

// I'm optional
void render() override{
// I run during every frame.
}
};

Engine Properties:

The engine can create with default parameters, but custom parameters are available. As seen below:

// I create a game with the name "Game Name", a width of 500, and height of 300
Game() : Phoenix::Engine( {"Game Name", 500, 300} ) {};

Extendable Functions:

create()

Function run on the creation of the engine

render()

Function run during every render frame
